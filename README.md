# planning_poker

## 1 - Lancer le backend
Linux
```
cd backend
python -m venv env
source env/bin/activate
pip install -r requirements.txt
python manage.py runserver
 ```

Windows 
```
cd backend
python -m venv env
env/script/acivate
pip install -r requirements.txt
python manage.py runserver
```

## 2 - Lancer le frontend
```
cd frontend
npm install
npm run dev
```
